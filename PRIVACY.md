# Privacy Policy for Bristo Sales Prospector

This Privacy Policy describes how the Bristo Sales Prospector (the "Agent") handles user data.

## 1. Data Processing
The Agent is **stateless**. It does not store any user data, chat history, or lead lists. It only processes company names and prospect names provided in the chat to generate real-time research summaries.

## 2. Data Storage
No data is stored on our servers. All information processed is used immediately to generate a response and is then discarded.

## 3. Transparency
The Agent only accesses public company information to provide firmographics and strategic insights.
